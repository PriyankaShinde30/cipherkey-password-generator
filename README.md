# CipherKey Password Generator

CipherKey is a secure password generator built using HTML, CSS, and Vanilla JavaScript. It allows users to create strong passwords with different customization options while keeping everything completely client-side. No passwords are sent to a server or stored online.

## Features

- Generate secure random passwords
- Adjustable password length (4–64 characters)
- Include uppercase, lowercase, numbers, and symbols
- Exclude similar characters (O, 0, I, l)
- Exclude ambiguous symbols
- Prevent repeated characters
- Avoid sequential characters
- Option to include spaces
- Option to start passwords with a letter
- Password strength indicator
- Quick preset configurations
- Copy passwords to the clipboard
- Password history using Local Storage
- Delete individual passwords from history
- Light and Dark theme with saved preference
- Responsive design

## Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Clipboard API
- Font Awesome

Clone the repository:

```bash

```

Open the project folder and launch `index.html` in your browser.

No installation or dependencies are required.
